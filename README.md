# PO-Handoff 🤝

**Seamless context handoff for Product Owners** switching between Cursor, Claude, Gemini, ChatGPT & more.

Never lose your PRD sections, user stories, acceptance criteria, decisions, risks, or the "why" behind every choice again.

Built by **masterOfbacklog** — 

---

## The Problem

When I jump between AI tools while doing product work:
- I lose all context
- I waste tokens re-explaining decisions
- Momentum dies
- I end up repeating the same requirements

## The Solution

**PO-Handoff** lets you capture your entire product context in a clean structured format and hand it off instantly to any AI agent.

One agent → exports context  
Next agent → loads it perfectly and continues exactly where you left off.

## Features

- ✅ Captures full task state (PRD, User Stories, ACs, Priorities, Risks, Assumptions)
- ✅ Preserves decisions, trade-offs & failed approaches
- ✅ Clean structured JSON + ready-to-paste prompts for Cursor, Claude, Gemini
- ✅ Simple CLI to validate, export & import context
- ✅ Designed specifically for Product Owners & PMs (not just developers)
